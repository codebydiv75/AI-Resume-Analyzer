# 📄 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, and Google's Gemini API.

The application analyzes a resume against a job description and provides:
- Resume Match Score
- ATS Score
- Missing Skills Analysis
- Resume Improvement Suggestions
- Technical Skill Extraction
- 30-Day Career Improvement Roadmap

---

## 🚀 Features

### 📊 Resume Analysis
- Compare resume with a job description
- Generate Resume Match Score
- Generate ATS Score
- Identify strengths and weaknesses

### 🛠 Skill Extraction
- Extract technical skills from resumes
- Categorize skills into:
  - Programming Languages
  - Frameworks
  - Databases
  - Cloud Technologies
  - Tools

### 📈 Career Roadmap
- Generate a personalized 30-day improvement plan
- Suggest skills to learn
- Recommend projects to build
- Provide career development advice

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone <your-repository-url>
cd AI-Resume-Analyzer
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

Windows:

```bash
venv\Scripts\activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Create .env File

```env
GEMINI_API_KEY=your_api_key_here
```

### 5. Run Application

```bash
streamlit run app.py
```
